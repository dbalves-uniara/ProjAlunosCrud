Uma aplicação de cadastro de Alunos.
Ela conta com a adição de novos alunos, edição de alunos já cadastrados e exclusão de alunos


Para executar a aplição, baixe todos os arquivos. Após baixar, abra a pasta ProjAlunosCrud em um editor de código (exemplo: IntelliJ, Visual Studio Code), após isso abra o arquivo Main e execute-o. Assim que executar, abra um navegador (exemplo: Chrome, Edge, etc...) e digite http://localhost:8080 (Caso o seu localhost:8080 estejá em uso, abra a o arquivo application.properties e adicione uma linha com 'server.port=8081' ou qualquer outro numero que esteja disponivel, e mude o numero de seu localhost para o mesmo que você adicionou no arquivo) ao executar estes passos a aplicação deverá estar rodando conforme o esperado e você poderá adicionar, editar e excluir os seus Alunos
